Photonic Architecture

One Process.
Four Operations.

Matrix multiplication, nonlinear activation, memory, and weight updates — all performed by second-order parametric nonlinearities. No digital logic. No separate hardware per function.

How It Works

χ⁽²⁾ Does the Work

Three optical fields couple through the nonlinear polarization of the material. By controlling pump amplitude and phase-matching, the same interaction performs every operation a neural network requires.

Matrix Multiply
Sum-frequency generation + interference

Weights encoded in the pump; activations in the signal. SFG produces an output at the sum frequency proportional to pump × signal — one weighted multiply per physical pass. Optical interference across inputs completes the summation.

SFG · ωₛ = ω₁ + ω₂ · coherent superposition
Nonlinear Activation
Parametric saturation

Pump depletion causes conversion efficiency to saturate with input amplitude, producing a nonlinear optical transfer function. The activation emerges from the physics — no separate digital step, no additional hardware.

Pump depletion · saturating transfer function
Memory
Optical state retention

Activation states are retained in the optical domain through parametric interactions, eliminating DRAM round-trips between layers. The backward pass reads the same optical state written during the forward pass.

On-chip optical state · no DRAM
Weight Updates
In-place pump reconfiguration

Weights are encoded in the pump field. Updating a weight means updating the pump — amplitude, phase, or both. The interaction reconfigures in place, no external memory write required.

Pump field encodes weights · in-place update
Why This Architecture

The Properties That Make It Work

Second-order parametric nonlinearities have physical properties that map directly onto what neural network training requires.

Reversible

χ⁽²⁾ interactions are fully reversible. The same physical process runs forward for inference and backward for gradient computation — no separate backward-pass hardware needed.

Coherent

Phase is preserved throughout. Interference is exact and deterministic, making coherent superposition a reliable mechanism for summation across inputs.

Instantaneous

χ⁽²⁾ interactions have no response time — computation occurs at the speed of the optical pulse itself, not limited by carrier dynamics or clock cycles.

Lossless

Parametric processes conserve energy. No energy is dissipated in the nonlinear interaction — only at chip boundaries, making recycling effective.

Levu Innovations

What We Built

Three breakthroughs that make parametric optical computing practical.

01
Memory
All-Optical Memory

A solution to optical memory — a long-standing open problem in photonic computing. Activation states persist entirely in the optical domain, no O/E conversion required.

02
Training
All-Optical Weight Updates

Weight updates performed entirely in the optical domain — no electronic intermediary, no off-chip memory writes. The network trains at the speed of light.

03
Efficiency
Optical Energy Recycling

Waste optical energy from each operation is captured and recirculated. Fiber optics have effectively zero propagation loss, so recirculation dramatically reduces total energy draw.
